The SQL Query Formatter tool will help you to write readable and maintainable queries by applying the best practices.

For instance:

select name, surname, substr(date, 1, 4) "Year" from employees where id>100 order by name, surname desc;

will be replaced by:

SELECT
    name,
    surname,
    SUBSTR(date, 1, 4) "Year"
FROM
    employees
WHERE
    id > 100
ORDER BY
    name,
    surname DESC
;

Please end your queries with a semicolon.
